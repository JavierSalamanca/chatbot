Herencia: Código

public class NintendoCo {

      
       private Videojuego[] videojuegos;
       private ArticuloElectronicoDeConsumo[] articulosDeConsumo;
       private String locacion;
       private int añoCreacion;

public NintendoCo(){
            
             videojuegos = new Videojuego[7000];
             articulosDeConsumo = new ArticuloElectronicoDeConsumo[200];
             locacion = "Kioto, Japón";
             añoCreacion = 1889;
            
       }
}

Se está definiendo la clase NintendoCo, que se puede describir como una receta sin cantidades. Dice qué elementos debe tener la compañía NintendoCo pero no en qué cantidad.
Éste es el proceso de declarar el nombre de las variables y que tipo tendrán. El tipo de las dos primeras variables se declararán mas adelante; “String” es para las cadenas de letras e “int” que significa “integer” que es para los números enteros.
