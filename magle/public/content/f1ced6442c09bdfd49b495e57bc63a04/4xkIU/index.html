Herencia

  • La herencia es un mecanismo por el que se pueden crear nuevas clases a partir de otras existentes, es decir, es la derivación de una clase a partir de otra existente.
    • Heredando, y posiblemente modificando, y/o añadiendo métodos (funciones/operaciones/acciones).
    • Heredando y posiblemente añadiento atributos.
  • El objetivo es la reutilización del software desarrollado.
