Herencia


public class Vehiculo {
    
    private String patente;
    private String marca;
    private String modelo;
    
    public Vehiculo() {
        //Constructor
    }

}

public class Avion extends Vehiculo{

    private int numeroMotores;
    private int presionMaxima;
    
    public Avion() {
        //Constructor
    }
}


public class Bus extends Vehiculo{
    
    private int numeroEjes;
    private int capPasajSent;
    private int capPasajDePie;
    
    public Bus(){
        //Constructor
    }
}

  • La herencia, es la derivación de una clase a partir de otra existente

  • Hereda y posiblemente modifica o añade métodos pero NO elimina

  • Hereda y posiblemente añade atributos pero NO elimina

  • El objetivo es la reutilización del software desarrollado
